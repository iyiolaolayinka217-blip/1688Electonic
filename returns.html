Returns & Refunds

No-hassle returns for peace of mind. We understand that sometimes a part isn't the right fit. Our goal is to make the return process as efficient as possible.

Return Window

30 days from the date of delivery.

Condition

Items must be in original packaging. Static-sensitive components must remain in their unopened ESD (Electrostatic Discharge) bags.

RMA Process

Log into your account, select the order, and click "Request RMA" to receive your return shipping label.

Restocking Fee

A small 10% fee may apply to high-value items returned without a defect.

How to Return an Item

1

Request Return

Submit a return request through your account or contact support

2

Get Authorization

Receive RMA number and return instructions via email

3

Ship Item

Package securely and ship using provided return label

4

Receive Refund

Refund processed after item inspection and approval

Return Eligibility

Eligible for Return

  • Items in original condition
  • Unopened electronics (within 30 days)
  • Defective or damaged items
  • Wrong item received
  • Items with manufacturing defects
  • Accessories and components

Not Eligible

  • Items past 30-day window
  • Software and digital products
  • Consumables (inks, toners)
  • Custom or personalized items
  • Items with removed serial numbers
  • Items damaged by misuse
